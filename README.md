# Habits

Skills tell agents what to do. Habits help agents get better at doing it.

A Habit is a self-improving, context-optimized skill: a repeated workflow that
keeps routine mechanics out of active context, returns the next instruction only
when it is needed, and turns failures into structured recovery paths.

That is how human habits work. When you brush your teeth, you do not rehearse
every step. You do not stop to reason about how toothpaste gets onto the brush.
And if the water is off, that exception becomes conscious only when it matters.

Agent Habits bring the same pattern to AI workflows. The practiced parts of a
routine live in executable state, checks, ledgers, handoffs, and recovery
guidance. The agent stays focused on judgment while the Habit handles the
repeatable motion.

Agents are probabilistic systems, so context is not free. Every extra
instruction in the live prompt is both cost and risk: more tokens to spend,
more details to juggle, and more chances to miss the step that matters. Habits
shrink the live prompt by moving practiced protocol into durable state and
just-in-time guidance.

The loop is simple:

```text
run -> record -> recover -> evolve -> improve
```

The result is a skill that gets lighter to use over time: less checklist
management, fewer repeated protocol mistakes, clearer recovery when something
breaks, and reviewable improvements when the Habit learns from experience.
