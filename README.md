# Habits

Humans do not stay conscious of every step in a skill forever. With practice,
repeated actions become habits: reliable procedural memory that runs in the
background until judgment, adaptation, or intervention is needed.

Habits brings that same idea to coding agents. A Habit is a skill that has
learned enough structure to move repeated workflow mechanics out of the
agent's active context and into executable state, checks, and handoffs. The
agent keeps judgment; the Habit handles the practiced routine.

The goal is to make complex agent workflows feel less like long checklists and
more like learned behavior: smaller context load, fewer fragile protocol
steps, clearer recovery paths, and structured guidance when the routine needs
conscious attention again.
